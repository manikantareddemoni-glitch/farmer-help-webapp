# Farmer Help WebApp 🌾

A responsive web application built to support farmers with modern digital tools.

## Features

- 🌱 Sell crops online
- 🛒 Buy pesticides and farming products
- 🤖 AI chatbot for farming guidance
- 🌦 Weather updates
- 🌐 Multi-language support
- 📱 Fully responsive design

## Technologies Used

- React JSX
- CSS
- JavaScript
- SQLite / Firebase (optional)
- AI Chat Integration

## Future Improvements

- Live market prices
- Voice assistant
- Government scheme updates
- Disease detection using AI
